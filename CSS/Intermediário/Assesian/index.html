O que é o assesian?

Assesian é um sistema de autenticação de usuários baseado na sua dinâmica de digitação por meio do uso da inteligência artificial.

Como funciona?

Para que o sistema consiga formar um padrão de digitação, é necessário que o usuário digite algumas vezes repetidamente uma senha cadastrada em um loop de treinamento do sistema.
